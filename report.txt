# pd2-sudoku
執行檔案：(檔案都在資料夾中)
1 make
2./give_question > ques.txt
3./solve < ques.txt

Sudoku.cpp內分成三部分:
一、出題
  隨機出題:
  1.先設一基本盤面
  2.取亂數讓它鏡射、反轉、換排、換數字
  3.挖空

二、讀題
  用cin直接讀進題目

三、解題
  1.先備份一份題目
  2.用DFS法找答案
  (check用來檢查可以用的數字)
  3.檢查是否有多重解
  
class Sudoku
{
 public:
  Sudoku();
  void GiveQuestion(); //用來出題
  void ReadIn();  //用來讀題
  void Solve(); //用來解題
  static const int sudokuSize = 144;

 private:
        int map[sudokuSize]; //用來存讀到的題目
        int copy[sudokuSize]; //用來備份題目
	int R; //隨機亂數，用來隨機使原始題目翻轉
	bool checkUnity(int arr[]); //檢查每個直排與橫排
	bool checkCell(int arr[]); //檢查九宮
	int Ans[sudokuSize]; //用來存答案
	int Ans_b[sudokuSize]; //存第二組答案
	bool isCorrect(); //用來確認直排、橫排與九宮皆是對的
	bool Solution(); //遞迴填數字
	int A; //用來存答案個數
	bool check(int, int); //用來檢查每橫排、直排和九宮可以填的數字
	void DFS(int);  //用來找答案
	void dfs(int); //用來找第二種答案
};
